Техническое задание:
Разработать программу, позволяющую зашифровать и расшифровать текст с использованием шифра Цезаря

Программа должна поддерживать несколько режимов работы:
1)Шифровка текста с помощью ключа - пользователь вводит ключ
2)Расшифровка текста с помощью ключа - пользователь вводит ключ
3)Расшифровка текста с помощью Brute Force - перебором всех вариантов

Особенности:
-Программа должна открывать указанный пользователем файл с текстом
-Проводить с ним одну из указанных выше операций
-Программа должна создавать новый файл с результатом
-Обеспечивать обработку больших файлов(читаем через BufferedReader)
-Иметь валидацию входных данных
-Принимать и создавать файлы формата .txt

Опционально:
-Создать графический интерфейс
-Поддерживать статистический анализ текста
